This folder contains data obtained from high throughput microscopy assay. 

Data format for files *-GLOBAL_GROWTH_RATE.txt 
-----------------------------------------------

## Data format ##
Column 1 : Well_FieldOfView_ColonyNo
Column 2 : MIcrocolony growth rate (per hour)
Column 3 : Centroid position of the microcolony (x,y) co-ordinates 


Data format for files *-GLOBAL_GROWTH_RATE2.txt 
-----------------------------------------------

## Data_format ##
Column 1: Well_FieldOfView_ColonyNo
Column 2: Centroid position of the microcolony (x,y) co-ordinates 
Column 3 onwards : Micro-colony area over time 
                   NaN - No data
